# 🕰️ Time Travel Journalist Agent
**Build in APAC. Build for the world.**

## Overview
The **Time Travel Journalist** is an immersive AI agent built for the Google Cloud Gen AI Academy. It solves the problem of history feeling like a dry list of facts by transforming real-world data into vivid, first-person "Live Dispatches."

## Core Features
* **Multi-Agent Orchestration**: Chaining a factual **Historian Researcher** with a creative **Time Journalist** to ensure reports are both imaginative and fact-based.
* **Live Fact-Finding**: Integrated with the Wikipedia API via Langchain for real-time data retrieval.
* **Cloud Native**: Deployed on **Google Cloud Run** using the **Agent Development Kit (ADK)**.

## How it Works
1.  The user enters a historical era or event (e.g., "Paris, July 1789").
2.  The **Researcher Agent** investigates current real-world data and historical figures.
3.  The **Journalist Agent** uses that data to generate a realistic, immersive "Day in the Life" story.

## Technologies Used
* **Python & ADK**: Core logic and orchestration.
* **Google Vertex AI**: Powering the generative models.
* **Google Cloud Run**: Serverless hosting for the web interface.
* **Wikipedia API**: Live data source.
